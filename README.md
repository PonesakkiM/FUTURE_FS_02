## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/PonesakkiM/FUTURE_FS_02.git
cd FUTURE_FS_02

Run Backend

```bash
cd server
npm install
npm start
```


Run Frontend

```bash
cd client
npm install
npm start
```
